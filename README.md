# Markdown 常用语法速查表
## 说明
本速查表适用于技术文档编写，兼容 GitHub、Notion、VS Code 等主流平台

| 语法类型 | 语法格式 | 示例 | 渲染效果 |
|----------|----------|------|----------|
| **一级标题** | `# 标题` | `# Java 策略模式` | # Java 策略模式 |
| **二级标题** | `## 标题` | `## 核心角色` | ## 核心角色 |
| **加粗文本** | `**文本**` 或 `__文本__` | `**开闭原则**` | **开闭原则** |
| **斜体文本** | `*文本*` 或 `_文本_` | `*行为型模式*` | *行为型模式* |
| **加粗+斜体** | `***文本***` | `***核心思想***` | ***核心思想*** |
| **删除线** | `~~文本~~` | `~~废弃方法~~` | ~~废弃方法~~ |
| **单行代码** | `` `代码` `` | `` `pay(double amount)` `` | `pay(double amount)` |
| **多行代码块** | ```语言<br>代码<br>``` | ```java<br>public interface PaymentStrategy {}<br>``` | ```java<br>public interface PaymentStrategy {}<br>``` |
| **无序列表** | `- 内容` / `* 内容` | `- 策略接口<br>- 具体策略类` | - 策略接口<br>- 具体策略类 |
| **有序列表** | `数字. 内容` | `1. 定义接口<br>2. 实现策略` | 1. 定义接口<br>2. 实现策略 |
| **任务列表** | `- [ ] 未完成<br>- [x] 已完成` | `- [x] 编写示例<br>- [ ] 对比模式` | - [x] 编写示例<br>- [ ] 对比模式 |
| **超链接** | `[链接文字](地址 "标题")` | `[Java 文档](https://docs.oracle.com "Java 官方文档")` | [Java 文档](https://docs.oracle.com "Java 官方文档") |
| **图片** | `![描述](地址 "标题")` | `![UML 图](./uml.png "策略模式 UML")` | ![UML 图](./uml.png "策略模式 UML") |

| **分隔线** | `---` / `***` | `---` | --- |
| **转义字符** | `\符号` | `\# 不是标题` | \# 不是标题 |

## 使用提示
1.  所有语法符号（`#`、`*`、`-` 等）与文字之间需加 **一个空格**
2.  分段用**空一行**实现，换行需在行尾加 **两个空格**
3.  代码块指定编程语言可触发**语法高亮**
4.  任务列表仅部分平台支持（如 GitHub、GitBook）
