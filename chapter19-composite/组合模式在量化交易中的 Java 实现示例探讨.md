1.  **文件/目录系统**

2.  **组织架构/权限系统**

公司（根容器）包含部门（子容器），部门包含员工（叶子节点）；权限系统中，角色组（容器）包含子角色组或单个权限（叶子）。

*项目案例*：企业OA的组织架构展示、权限管理系统的角色分配功能。

3.  **菜单导航系统**

主菜单（容器节点）包含子菜单（子容器）或菜单项（叶子节点），客户端统一渲染菜单、处理点击事件。

*项目案例*：后台管理系统的侧边栏菜单、电商网站的顶部导航菜单。

4.  **图形界面（GUI）控件树**

*项目案例*：Swing/JavaFX的界面组件、前端Vue/React的组件树。
